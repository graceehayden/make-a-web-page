Lesson 3: Adding Style to HTML Using CSS

Avoiding Repetition

If programmers repeat the same code over and over in different places, any future changes will be harder to make, as they will have to go back and find each place it was used.

"div" and "class" labels allow for a universal application in multiple places with a particular label in a document. The styling for every element with that label can be done in one place.

CSS

CSS, or cascading style sheets, is styling done in a separate document that is linked to on the index page. It allows for non-redundancies by the assignment of classes, etcetera.
